# AegisCAD

AegisCAD – experimental full CAD/FEA/Assemblies/CAM/Drafting environment built on Qt6 + OpenCascade.

This repo skeleton περιέχει:
- CAD πυρήνα (Sketch, FeatureTree, FeatureOps, IO)
- Assemblies (mates, joints, solver)
- CAM module
- Drafting/2D drawings
- Analysis/FEA stubs (CalculiX backend)
- Local Aegis AI helper (rule-based)
- Reverse engineering hooks
- Performance helpers (LOD / perf profiling)
- Python scripting bridge
